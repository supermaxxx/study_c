[01] 外部传入值并读取
1) scanf    //这个遇到空格就断
#include <stdio.h>
int main()
{
  char name[20];
  printf("Please enter your name: ");
  scanf("%s",name);
  printf("Your name is: %s\n", name);
  return 0;
}
2) gets    //这个很危险，容易溢出，一般不用

  gets(name);
...
3) fgets    //使用这个函数，比较安全
...
  fgets(name,20,stdin);

[02] switch...case语法
#include <stdio.h>
main()
{
  int in;
  char *out;
  scanf("%d",&in);
  switch(in)
  {
    case 0:out="success";break;
    case 1:out="fail";break;
    default:printf("data error");break;
  }
  printf("%s",out);
}

[03] 函数
*函数的返回值类型必须和return返回值类型一致
*如果函数没有返回值，指定为void
*C语言中，函数返回值的类型可以省略，默认为int
例1)计算2个数的平均值
#include <stdio.h>
double average(double x, double y)
{
  return (x+y)/2;
}
int main()
{
  double a = 4.47;
  double b = 9.83;
  double avg = average(a,b);
  printf("%g 和 %g的平均数: %g\n",a,b,avg);
  return 0;
}
/*
result:
4.47 和 9.83的平均数: 7.15
*/
